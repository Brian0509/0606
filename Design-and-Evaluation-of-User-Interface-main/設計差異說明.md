> 使用者介面設計與評鑑 · 訪談任務二｜Group 1：張心洧・林彥銓・張宇睿
> 本文件對應作業「項目二：功能與特色的差異 × 與訪談結果的連結」，可直接作為 6/10 報告大綱。
> 兩個版本皆以**原始版本（第一版遊戲簡介）為基礎**調整，原始檔完全保留。

## 一、設計原則總覽

| | 男生版本 | 女生版本 |
|---|---|---|
| 核心取向 | 功能 / 數據導向 | 美觀 / 簡潔導向 |
| 配色 | 加深的藍 / 紫，**減少漸層**、改用單色，便於分類辨識 | 維持原本藍 / 紫，但走柔和、和諧、留白 |

## 二、男生版本：強調的功能（對應訪談）

1. **移除「非按鈕」元件的滑鼠特效**
   - 訪談：男生反映「有些不是按鈕的元件，滑鼠移上去會有特效，讓人以為是按鈕、以為網頁故障」。
   - 調整：資訊卡、步驟卡、模式卡、特性列、評分表列等純展示元素**一律取消浮起 / 投影 / 變色 hover，並改回一般游標**；也**移除舊版的「👆可點擊」角標**。真正可點擊的連結卡片才保留單純浮起回饋。

2. **進度條維持「閱讀進度＋百分比」**
   - 訪談：男生反映「進度條像讀取條、會空等它跑完」。
   - 調整：加底色軌道、右上角顯示「閱讀進度 ◯%」，捲到底轉綠色 100%。

3. **字體縮放範圍擴大**
   - 訪談：男生反映「縮放範圍太小、體感沒差」。
   - 調整：由 `85→116%`（±15%）擴大為 **`80→150%`（−20%～+50%）**。

4. **介紹文案放大 ＋ 大數據標語帶**
   - 訪談：男生（N=3）認為「大量數據（如 70 萬次下載）能更了解產品」。
   - 調整：知識王 / Kahoot 兩頁的介紹文案放大，並於 Hero 下方新增**大數據標語帶**（兩頁模板一致）：
     - 知識王：70萬+ 下載、10秒 限時、6大 領域、App Store No.1
     - Kahoot：3億+ 用戶、200+ 國家、2013 創立、90M+ 月活躍課堂

5. **站內搜尋（新功能）**
   - 訪談延伸：男生重視功能性與效率。
   - 調整：導覽列新增**站內搜尋欄**，輸入關鍵字會高亮本頁所有相符文字並逐一跳轉，類似大型校園官網。快捷指南同步**移除失效的 Shift+D**，改列站內搜尋說明。

6. **減少漸層 ＋ 比較表紅/黃/綠**
   - 訪談：男生（N=4）視「顏色為分類工具」。
   - 調整：Logo、按鈕、頁籤等改用**單色**（減少漸層）；「遊戲特性比較表」改用 **綠＝好、黃＝普通、紅＝差、灰＝中性資訊**，好壞一眼分辨。

## 三、女生版本：強調的功能（對應訪談）

1. **配色維持藍 / 紫、走柔和和諧**
   - 訪談：女生（N=4）「在意主視覺整體和諧、美觀優先」。
   - 調整：沿用原本藍 (#2196f3) / 紫 (#9c4dff)，搭配柔和淺底、圓潤卡片、輕陰影與更多留白，營造一致協調的視覺。

2. **知識王 / Kahoot 著重遊戲本身**
   - 訪談：女生（N=3）「密密麻麻的數據標語抓不到重點，偏好精簡」。
   - 調整：兩頁的介紹文案**移除大數據標語**（不放數字、不放標語帶），改以遊戲體驗與玩法為主軸描述（兩頁模板一致）。

3. **Q&A 改為整頁真人客服**
   - 訪談：女生（N=3）「看完 FAQ 仍希望有真人客服提供個人化回覆」。
   - 調整：Q&A 分頁**移除原本的常見問題與提問表單**，整頁改為**真人客服對話介面**（線上狀態、快捷提問、即時對話框）。其他頁面不再出現浮動客服按鈕。

4. **降低資訊擁擠、移除快捷鍵區塊**
   - 加大間距與行距、柔化卡片；首頁移除快捷功能指南（女性會略過）。

## 四、訪談發現 → 設計調整（總表）

| 訪談發現 | 版本 | 對應調整 |
|---|---|---|
| 非按鈕有特效→以為故障 | 男 | 資訊卡全面移除 hover 特效、移除「可點擊」角標 |
| 進度條像讀取條、空等 | 男 | 進度條加軌道＋「閱讀進度 %」 |
| 字體縮放範圍太小 | 男 | 範圍擴大為 −20%～+50% |
| 數據豐富有助了解產品 | 男 | 文案放大＋大數據標語帶 |
| 重視功能 / 效率 | 男 | 新增站內搜尋（高亮＋跳轉） |
| 顏色＝分類工具 | 男 | 減少漸層、比較表改紅/黃/綠 |
| 顏色＝整體和諧、美觀優先 | 女 | 維持藍紫、柔和留白和諧風 |
| 數據標語抓不到重點 | 女 | 移除大數據標語、著重遊戲介紹 |
| 看 FAQ 後仍想要真人客服 | 女 | Q&A 改為整頁真人客服對話 |
| 對快捷鍵直接略過 | 女 | 移除快捷鍵區塊 |

## 五、模板一致性

- 男版：知識王頁 ＝ Kahoot 頁（同樣有放大文案＋大數據標語帶＋相同結構）。
- 女版：知識王頁 ＝ Kahoot 頁（同樣著重遊戲介紹、無標語帶、相同結構）。
- 男女之間可不同（依訪談各自最適化）。
