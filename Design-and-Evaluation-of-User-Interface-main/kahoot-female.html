⚡玩法特色

01

🎮 主持人出題

主持人在大螢幕顯示題目，參與者使用手機輸入 Game PIN 加入，無需帳號即可參與，入門門檻極低，適合課堂即時使用。

02

⏱️ 限時搶答

每題有倒數計時條，最快且正確的答題者獲得最多分數。答題結果即時顯示正確率與排行榜，競爭氣氛緊張刺激。

03

📊 即時回饋與分析報告

遊戲結束後提供詳細答題報告，教師可分析學生學習狀況，有效評估學習成效，並針對弱點進行教學調整。

🎮遊戲模式

🏟️

Classic 競技模式

傳統多人搶答，即時排行榜讓競爭氣氛緊張刺激，適合課堂複習與知識測驗。

👨‍👩‍👧‍👦

Team Mode 團隊模式

分組競賽，培養團隊合作精神，特別適合企業培訓與課堂分組教學活動。

📝

Assign 作業模式

老師指派 Kahoot 作為家庭作業，學生自行完成，系統自動統計答題報告。

🛤️玩家體驗旅程課堂多人搶答流程

01

建立題組

主持人選擇或自建題組，按下開始後系統產生 6 位數 Game PIN。

02

掃碼加入

輸入 PIN 或掃 QR Code 進入等待大廳，不需帳號立即加入。

03

大螢幕出題

題目呈現在主持人螢幕，參與者手機只顯示四色選項，簡潔清晰。

04

倒數搶答

計時條由綠轉紅，答題後立即顯示對錯與全場正確率，熱烈搶答。

05

報告下載

排行榜公布後可下載完整答題報告，掌握每位參與者的作答狀況。

📍適合使用情境

📚

課堂即時互動測驗

老師在大螢幕投影題目，學生用手機搶答，不需帳號即可加入，課堂氣氛瞬間活絡起來。

📊

課後學習成效評量

Assign 作業模式讓老師指派題組，系統自動統整全班答題報告，快速掌握每位學生的學習狀況。

🏢

企業培訓與研習活動

Team Mode 支援分組競賽，主持人即時掌握全場答題狀況，遊戲化形式讓培訓內容更容易吸收。
